1.html element and tag the same thing?
no
HTML element:-
html element contains start tag,content and end tag.it is called as html element.
EX:- <HTML>is the opening tag, any content and </HTML>is the closing tag.

HTML tag:-
html tag contains only start tag and end tag.it is called as html tag.
EX:- <HTML>is the opening tag and </HTML>is the closing tag.

6.what is the 'class' attribute in html?with example.
class attribute:-
The class attribute specifies one or more classnames for an element.
The class attribute is mostly used to point to a class in a style sheet.
The class attribute can be used on any HTML element.

7.what is the difference between the 'id' attribute and 'class' attribute of html element ?with example.

id:-
The id attribute specifies a unique id for an HTML element.
The value of the id attribute must be unique within the HTML document.
The id attribute is used to point to a specific style declaration in a style           sheet.

class:-
The class attribute specifies one or more classnames for an element.
The class attribute is mostly used to point to a class in a style sheet.
The class attribute can be used on any HTML element.

11.what is the difference between a block level element and an inline element>

block element:-
A block-level element always starts on a new line, and the browsers automatically add some space (a margin) before and after the element.
A block-level element always takes up the full width available.
