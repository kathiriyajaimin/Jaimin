Form

will helps to send client data to the server

there are tow methods for sending data to the server

input type submit will helps to send data to the server

input name attribute is responsible for sending data to the server

  1. get
  2. post

GET

get is a default method for form

get will send data using URL

get less secure because it will send data to server by URL

by using ? (query string ) it will appending data to the URL

file:///C:/xampp/htdocs/WD/17DecWD-FE_TTS8/HTML/24form/formintro.html?username=test&password=test

get method has limitation of sending data to server is 2048 charector only 1024bytes

get method not support file uploading

POST

we can apply post method to the form by method attribute of form tag

Post will not send data using URL

POST secure because it's not using' URL for sending data

POST method no limitation

POST method support file uploading

for uploading file we need to add enctype="multipart/form-daa" attribute in form tag
